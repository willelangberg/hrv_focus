HRV Focus Monitor

Predicting cognitive fatigue using HRV and behavioral signals

About the Project

HRV Focus Monitor is a non-commercial, early-stage research project aimed at predicting cognitive fatigue through biometric signals like heart rate variability (HRV), stress, and sleep data, leveraging the Garmin Health API. The Python-based system is currently tailored to single-user experimentation, optimized for scalability and broader potential usage.

Why Garmin?

Garmin devices provide rich biometric data essential for real-time analysis of stress, recovery, and cognitive performance. HRV and heart-rate insights from Garmin wearables specifically support detailed trend analysis crucial for precise fatigue prediction during cognitively intensive activities.

How It Works

  • Data Collection: Garmin Health API delivers HR, RR intervals, stress, and sleep data in structured JSON format.
  • User Input: Daily subjective ratings on focus, energy, and fatigue complement biometric data.
  • Analysis and Predictions: Python models assess correlations to predict cognitive productivity dips proactively.
  • Future Goals: Integration with interactive dashboards and broader user accessibility.

Privacy & Compliance

Data security and user privacy are central to the HRV Focus Monitor project. All collected data remains local, encrypted, and strictly confidential. The application adheres fully to GDPR regulations and best practices in personal data management.
