HRV Focus Monitor

Project Purpose:

HRV Focus Monitor is an early-stage research project designed to help predict cognitive fatigue using biometric signals such as heart rate variability (HRV). The system is currently used by the developer only and is intended for personal productivity optimization.

Data Usage

The application uses the following data provided by Garmin (with user consent):

  • Heart Rate
  • HRV (e.g., RMSSD)
  • R-R Intervals
  • Stress and Sleep Metrics (optional)

Privacy Policy

All data is collected with explicit user consent and used solely for real-time feedback on mental fatigue and focus. The data is:

  • Stored locally and securely (encrypted when applicable)
  • Never shared with any third parties
  • Deleted immediately upon user request
