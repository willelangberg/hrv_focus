HRV Focus Monitor

Project Purpose:

HRV Focus Monitor is a personal productivity tool that uses Garmin's Health API to track heart rate variability (HRV) in real time. The goal is to help users recognize early signs of cognitive fatigue and optimize their work sessions for better focus and performance.

Data Usage

The application uses the following data provided by Garmin (with user consent):

  • Heart Rate
  • HRV (e.g., RMSSD)
  • R-R Intervals
  • Stress and Sleep Metrics (optional)

Privacy Policy

All data is collected only with explicit user consent and is used solely for the purpose of providing feedback on focus and fatigue. The data is:

  • Stored securely with encryption.
  • Never shared with third parties.
  • Deleted immediately upon user request.

Users can revoke access at any time via Garmin Connect settings.

This project follows GDPR guidelines and takes user privacy seriously.
