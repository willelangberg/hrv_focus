HRV Focus Monitor

Predicting cognitive fatigue using HRV and behavioral signals

About the Project

HRV Focus Monitor is a non-commercial, early-stage research project designed to predict cognitive fatigue using biometric signals such as enhanced beat-to-beat interval (RR), heart rate, stress, and sleep data — all sourced via the Garmin Health API. The system is developed in Python and currently serves a single-user (the developer) in a local-only setup.

How It Works

  • Data Collection: Garmin wearables provide HR, RR interval, stress, and sleep data in JSON format via the Health API.
  • Behavioral Inputs: The user submits daily self-ratings on focus, energy, sleep quality, and subjective fatigue.
  • Local Processing: A Python-based model analyzes correlations and predicts upcoming focus dips.
  • Outputs: Currently outputs local logs and graphs. Future versions may integrate a dashboard (e.g., Streamlit).

Data & Privacy

All data is collected with explicit consent and used solely for real-time feedback and research. The application does not share, sync, or transmit any user data. Data is stored locally and encrypted. The project complies with GDPR principles.
